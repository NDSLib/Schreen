# Requirement
 
read `requiments.txt`.
 
# Installation
  `python3.7.7`&`python3.6.5`で検証済み
```bash
pip install -r requirements.txt
```
 
# Usage
  
```bash
python app/main.py
```
 
# Note
`pyinstaller`で`.app`化するときは、`app/DEBUG.py`の`DEBUG=True`を`DEBUG=False`にしてください。
